# Classic Mania — 개발 로드맵

> 클래식 음악 커뮤니티 플랫폼의 사용자 유입과 리텐션을 극대화하기 위한 단계별 로드맵

---

## 현재 프로젝트 요약

Classic Mania는 클래식 음악 애호가를 위한 커뮤니티 플랫폼으로, 다음 핵심 기능을 보유하고 있습니다:

- **공연 탐색** — KOPIS API 연동, 필터/검색/지도 뷰
- **아티스트 디렉토리** — 작곡가(Open Opus), 지휘자, 연주자 탐색 및 팔로우
- **커뮤니티 게시판** — 카테고리별 게시글/댓글 CRUD
- **뉴스 피드** — 네이버 뉴스 API 연동
- **마이페이지** — 작성 글, 팔로우 아티스트 관리

---

## Phase 1: 검색 유입 & 바이럴 기반 구축 (1~2개월)

사용자가 검색 엔진과 SNS를 통해 자연스럽게 유입되도록 기반을 다지는 단계입니다.

### 1.1 SEO 최적화
- Next.js `metadata` API 활용한 동적 메타태그 생성 (공연명, 아티스트명 기반)
- `sitemap.xml` / `robots.txt` 자동 생성
- Open Graph / Twitter Card 이미지 자동 생성 (공연 포스터 + 타이틀 합성)
- 구조화된 데이터(JSON-LD) 적용 — `MusicEvent`, `Person`, `Article` 스키마

### 1.2 공연 리뷰 & 평점 시스템
- 별점(1~5) + 텍스트 리뷰 기능 추가
- 공연 상세 페이지에 리뷰 섹션 통합
- 리뷰 기반 평균 평점 표시 → 검색 결과에 별점 노출 (리치 스니펫)
- "베스트 리뷰" 선정 로직

### 1.3 SNS 공유 기능 강화
- 공연/아티스트/게시글별 공유 버튼 (카카오톡, X, 페이스북, 링크 복사)
- 공유 시 OG 이미지가 포함된 미리보기 카드 자동 생성
- "이 공연에 관심 있는 친구에게 공유하기" CTA

## Phase 2: 사용자 참여도 & 리텐션 강화 (2~3개월)

유입된 사용자가 머물고 다시 방문하도록 만드는 핵심 기능들입니다.

### 2.1 개인화 추천 시스템
- 팔로우한 아티스트 기반 공연 추천
- 관람 이력 / 리뷰 데이터 기반 "당신을 위한 공연" 섹션
- 좋아하는 시대(Epoch) 기반 작곡가/작품 추천
- 알림 기능: 팔로우 아티스트의 새 공연 등록 시 푸시 알림

### 2.2 공연 캘린더 & 알림
- 월별/주별 캘린더 뷰 (내가 관심 있는 공연 한눈에 보기)
- Google Calendar / Apple Calendar 연동 (`.ics` 내보내기)
- 공연 D-day 알림 (브라우저 Push Notification / 이메일)
- "오늘의 공연" 위젯

### 2.3 실시간 커뮤니티 기능
- 공연 실황 채팅 — 공연 당일 실시간 감상 공유
- 게시판 댓글 실시간 알림 (WebSocket 기반)
- "지금 이 공연 보는 사람" 실시간 카운터
- 이모지 리액션 (게시글/댓글에 다양한 반응)

### 2.4 유저 프로필 & 활동 기록
- 공개 프로필 페이지 (닉네임, 관람 이력, 리뷰, 팔로우 아티스트)
- 활동 배지 시스템 (첫 리뷰, 10회 관람, 베스트 리뷰어 등)
- 관람 통계 대시보드 (장르별, 월별, 지역별 — Recharts 활용)
- 팔로워/팔로잉 기능

## Phase 3: 콘텐츠 차별화 & 가치 창출 (3~4개월)

다른 플랫폼에서 찾기 어려운 고유한 콘텐츠와 경험을 제공합니다.

### 3.1 클래식 음악 학습 허브
- 작곡가별 대표곡 가이드 (시대별, 난이도별)
- "클래식 입문자를 위한 추천 플레이리스트" 큐레이션
- 작품 해설 콘텐츠 (곡의 배경, 구조, 감상 포인트)
- YouTube 영상 큐레이션 — 추천 연주 영상 모음

### 3.2 아티스트 인터뷰 & 칼럼
- 국내 연주자/지휘자 인터뷰 콘텐츠
- 전문가 칼럼 시스템 (공연 비평, 음반 리뷰)
- 에디터 추천 공연 ("이번 주 꼭 봐야 할 공연")
- 유저 기고 시스템 — 커뮤니티 유저가 칼럼 작성

### 3.3 공연장 가이드
- 공연장별 상세 정보 (좌석 배치도, 좌석별 시야/음향 리뷰)
- "이 공연장 근처 맛집/카페" 가이드
- 공연장 방문 후기 (커뮤니티 연동)
- 접근성 정보 (주차, 대중교통, 장애인 편의시설)

### 3.4 음반/음원 리뷰 섹션
- 클래식 음반 데이터베이스 (MusicBrainz API 연동)
- 음반 리뷰 & 평점
- "같은 곡 다른 연주" 비교 리뷰
- 스트리밍 서비스 링크 연동 (Spotify, Apple Music, YouTube Music)

## Phase 4: 소셜 & 오프라인 연결 (4~6개월)

온라인 커뮤니티를 오프라인 경험과 연결하여 강력한 네트워크 효과를 만듭니다.

### 4.1 동행 구하기 기능
- "이 공연 같이 갈 사람?" 모집 게시판
- 지역/날짜/공연 기반 매칭
- 1:1 메시지 (DM) 기능
- 동행 후기 & 신뢰도 시스템

### 4.2 소모임 & 이벤트
- 클래식 감상 모임 개설/참여
- 오프라인 번개 (공연 후 뒤풀이, 음반 감상회 등)
- 모임 캘린더 & 참석자 관리
- 온라인 감상 모임 (화상 채팅으로 함께 음악 감상)

### 4.3 티켓 중고거래 & 양도
- 안전한 티켓 양도 플랫폼 (에스크로 시스템)
- 공연별 양도 게시판
- 가격 비교 기능
- 사기 방지 시스템 (인증된 티켓 확인)

---

## Phase 5: 수익 모델 & 확장 (6개월~)

플랫폼의 지속 가능성을 확보하는 수익 모델을 구축합니다.

### 5.1 제휴 & 파트너십
- 공연 기획사 제휴 — 할인 쿠폰 / 선예매 기회 제공
- 공연장 파트너십 — 좌석 추천 & 직접 예매 연동
- 음반사 제휴 — 신보 소식 & 프로모션

### 5.2 프리미엄 기능 (선택적)
- 광고 제거
- 공연 알림 고급 설정 (맞춤 조건)
- 전문가 칼럼 전용 콘텐츠
- 프로필 커스터마이징

### 5.3 모바일 앱 (PWA → 네이티브)
- PWA 우선 출시 (설치 유도 배너)
- React Native 네이티브 앱 전환
- 푸시 알림, 오프라인 지원, 위치 기반 공연 추천
